## ğŸ“ Customization

### Adding Events to Calendar
The events calendar is dynamically generated. To add event dates with highlights, update the JavaScript in `script.js`.

### Updating Sponsor Information
Edit the sponsor cards in the "Our Sponsors" section in `index.html`.

### Changing Colors
Modify the primary color throughout by updating CSS variables:
- Main Purple: `#762d78`
- Dark Purple: `#5a1f5c`
- Light Backgrounds: `#f5f5f5`, `#f0f0f0`

### Updating Team Members
Edit the team section in `team.html` with new team member information.
